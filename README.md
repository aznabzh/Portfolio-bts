# Portfolio BTS SIO SLAM - E5

## Overview
This project is a portfolio website created for the BTS SIO SLAM E5 oral exam.

Its purpose is to present:
- projects completed during the 2 years of BTS
- competencies worked on through those projects
- proofs associated with projects and competencies
- technology watch
- a simple contact page

The website is intended to be used as a presentation support during the oral exam.

---

## Project Philosophy
This project follows a simple rule:

**Preserve the validated visual direction, improve the internal structure.**

The current validated v0 mockup is the visual reference.
The implementation should progressively become:
- cleaner
- more maintainable
- more data-driven
- easier to extend

without unnecessarily changing the validated design.

---

## Main Principles
- multi-page website
- primarily static
- no backend
- no database
- no authentication
- no admin dashboard
- no CMS
- structured local content
- reusable components
- centralized visual rules
- optimized for oral presentation

---

## Main Pages
- Home
- Projects
- Project detail
- Competencies
- Technology watch
- Contact

---

## Key Functional Goal
The portfolio must clearly support the relationship between:

**project -> competency -> proof**

This relationship is central for the E5 oral exam.
